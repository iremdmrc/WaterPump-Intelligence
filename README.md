# HydroInsight 🚰

Data preprocessing and analysis project on the Tanzania water pump dataset.
This project focuses on cleaning, transforming, and visualizing data for further machine learning tasks.

---

## 📊 Dataset

Download dataset from:
https://www.drivendata.org/competitions/7/pump-it-up-data-mining-the-water-table/data/

This dataset is from the **DrivenData "Pump it Up: Data Mining the Water Table" competition**.

It contains real-world data about water pumps in Tanzania, including their condition:

* Functional
* Needs repair
* Non-functional

---

## ⚙️ Project Workflow

The following preprocessing steps were applied:

* Data integration (merged features and labels)
* Data cleaning (handled missing values, removed duplicates)
* Feature selection (used numerical features to optimize performance)
* Data normalization (scaled values between 0 and 1)
* Dimensionality reduction using PCA
* Data visualization (PCA scatter plot and histogram)

---

## 🚀 How to Run

1. Download the dataset from the link above
2. Open the notebook in **Google Colab**
3. Upload the CSV files
4. Run all cells
